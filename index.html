Discussion

The difficulties of this project:
  • To take the good informations that you need to show.

    • In data visualization you need to select the good data for sharing the information that you really want to show. For this project i tried almost three websites to find the period that i needed and the data in "xls,xlsx or csv". Some websites provide these information in the txt. format ( really complecated to manage this kind of data for me). With more reflexion i think i had to use the data the provides by the yahoo finance website as we've seen in course.

  • Traiting the Data : use the good packages

    • For a good visualization of your data you need to know which plot packages to use.

  • Merge different Data : find the good axis and be sure you have the same common dates for examples(for having a long historical dataset)

    • For a good data analysis you need to have a large data set period, if you want to compare different data set between them you have to merge them on the good dates and verify that the events you want to show and comment must be in the common period. In this case the scale of the differents index is not the same, if you want to plot them in the same graph the results cool be aweful. If you have too much informations to show it could be worse.
